Este repositório reúne todos os arquivos e materiais desenvolvidos para o protótipo do robô Otto Ninja, concebido como um modelo didático e experimental para simular atividades em um canteiro de obras, com foco em funções análogas às de uma retroescavadeira, de forma simplificada e automatizada.

O projeto foi desenvolvido com o objetivo de integrar modelagem física, controle lógico, simulação e documentação técnica, servindo tanto como protótipo funcional quanto como base inicial para futuros projetos acadêmicos e experimentais na área de automação, robótica e engenharia.

**Objetivo do Projeto**

O principal objetivo do protótipo é:

Desenvolver um robô capaz de simular operações básicas de um canteiro de obras, inspiradas no funcionamento de uma retroescavadeira;

Aplicar redes de Petri para modelar, analisar e validar o comportamento do sistema;

Integrar conceitos de robótica, sistemas a eventos discretos, modelagem 3D e planejamento de projetos;

Disponibilizar um material completo e reutilizável, permitindo que outros estudantes ou pesquisadores possam replicar, adaptar e evoluir o protótipo.

**Conteúdo do Repositório**

O repositório contém:

**-->** *Protótipo do Robô Otto Ninja:*

Estrutura e concepção do robô baseada na plataforma Otto Ninja;

Adaptações mecânicas e funcionais para simular movimentos e tarefas relacionadas a uma retroescavadeira.

**-->** *Redes de Petri*

Modelos desenvolvidos em:

HPsim

CPN IDE

Arquivos contendo:

Definição de lugares, transições e marcas;

Guards, variáveis e tokens inteiros;

Simulações validadas do comportamento do robô;

Utilização das redes de Petri para modelar a lógica de controle, sequência de operações e estados do sistema.

**-->** *Modelagem 3D – Fusion 360*

Arquivos de modelagem 3D desenvolvidos no Fusion 360, incluindo:

Estrutura do robô;

Componentes mecânicos;

Peças adaptadas ou projetadas especificamente para o protótipo;

Modelos prontos para visualização, edição e impressão 3D, quando aplicável.

**-->** *Peças e Componentes*

Relação de todas as peças necessárias para a construção do protótipo, incluindo:

Componentes mecânicos;

Estruturas impressas;

Elementos eletrônicos utilizados;

Organização dos arquivos de forma a facilitar a replicação do projeto físico.

**-->** *Documentação Técnica*

Relatório completo do trabalho, contendo:

Fundamentação teórica;

Metodologia;

Desenvolvimento do projeto;

Resultados e conclusões;

Slides de apresentação, desenvolvidos em Overleaf (LaTeX), prontos para uso acadêmico;

Documentos organizados para facilitar entendimento, avaliação e reapresentação do projeto.

**Finalidade para Futuros Usuários**

Este repositório foi estruturado para servir como:

- Base inicial de estudo para projetos de robótica e automação;

- Guia prático para construção e execução de um protótipo semelhante;

- Referência para aplicação de redes de Petri em sistemas robóticos;

- Ponto de partida para melhorias, adaptações e expansões do projeto, tanto em nível de hardware quanto de software.

Qualquer pessoa que tenha acesso a este repositório poderá compreender o projeto desde sua concepção até a execução, utilizando os arquivos disponíveis como suporte para aprendizado e desenvolvimento.
